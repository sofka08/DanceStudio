# DanceStudio
Мобильное приложение для автоматизации записи на занятия в танцевальной студии.

## О проекте

**Автоматизация работы танцевальной студии** – это мобильное приложение, созданное для упрощения взаимодействия клиентов с танцевальной студией. Платформа позволяет клиентам просматривать каталог направлений и тренеров, записываться на занятия, оставлять отзывы, а администратору — управлять контентом и отслеживать статистику.

### Основные возможности

- Регистрация и авторизация пользователей (клиент, администратор, гость)
- Просмотр каталога танцевальных направлений и тренеров
- Просмотр групп занятий с расписанием и свободными местами
- Бронирование мест в группах
- Отмена записи
- Просмотр истории посещений занятий
- Написание и просмотр отзывов с рейтингом
- Административная панель (управление направлениями, тренерами, группами)
- Просмотр статистики (клиенты, записи, популярные тренеры и направления)
- Модерация отзывов

## Установка и запуск

### Требования

- Android Studio (последняя версия)
- JDK 11 или выше
- Устройство или эмулятор с Android 8.0+
- Аккаунт Firebase

### Шаги для запуска
1. Перейти по QR-коду на Яндекс.Диск и скачать APK-файл

## Структура проекта

```
app/src/main/java/com/example/dancestudio/
│
├── activities/
│   ├── LoginActivity.java          # Вход в систему
│   ├── RegistrationActivity.java   # Регистрация
│   ├── MainActivity.java           # Главный экран
│   ├── ClientMenuActivity.java     # Меню клиента
│   ├── AdminMenuActivity.java      # Меню администратора
│   ├── DirectionsActivity.java     # Список направлений
│   ├── TrainersActivity.java       # Список тренеров
│   ├── GroupsActivity.java         # Список групп
│   ├── BookingActivity.java        # Бронирование
│   ├── MyBookingsActivity.java     # Мои записи
│   ├── ReviewsActivity.java        # Отзывы
│   ├── AddDirectionActivity.java   # Добавление направления (админ)
│   ├── AddTrainerActivity.java     # Добавление тренера (админ)
│   ├── AddGroupActivity.java       # Добавление группы (админ)
│   └── StatisticsActivity.java     # Статистика (админ)
│
├── adapters/
│   ├── DirectionAdapter.java       # Адаптер для направлений
│   ├── TrainerAdapter.java         # Адаптер для тренеров
│   ├── GroupAdapter.java           # Адаптер для групп
│   ├── BookingAdapter.java         # Адаптер для записей
│   └── ReviewAdapter.java          # Адаптер для отзывов
│
└── model/
    ├── User.java                   # Модель пользователя
    |── Direction.java              # Модель направления
    ├── Trainer.java                # Модель тренера
    ├── Group.java                  # Модель группы
    ├── Booking.java                # Модель бронирования
    └── Review.java                 # Модель отзыва
```
